# Brazil Rental Price Analysis

## Project Overview

This project explores the factors that influence residential rental prices in major Brazilian cities through Exploratory Data Analysis (EDA).

The objective was to identify which property characteristics have the strongest relationship with rental values and generate insights that may support future predictive models.


## Dataset

The dataset contains information about residential properties available for rent in Brazil, including:

- City
- Area (m²)
- Bedrooms
- Bathrooms
- Parking Spaces
- Floor
- HOA Fee
- Property Tax
- Fire Insurance
- Furnished Status
- Pet Acceptance
- Rental Price


## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab


## Main Findings

- São Paulo presented the highest average rental prices.
- Property area showed strong correlation with rent value (0.66).
- Bathrooms showed the strongest correlation with rent value (0.67).
- Bedrooms and parking spaces also demonstrated relevant influence.
- Furnished properties tend to have higher rental prices.
- Properties that accept pets showed slightly higher average rents.
